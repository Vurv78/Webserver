Features

API Status Description Args
"/digscreen" Online Returns comma separated RRRGGGBBB formatted Image Data Url,Res,Version[O]
"/query" Online Sends query to discord webhooks. Max 2000 chars. Returns error or "success". Webhook,Content,Name[O],Avatar[O]
"/ytdl" Online Returns url to audio from a youtube video using python-ytdl. Defaults to m4a. Url,Audioformat[O]
"/speedreq" Online Returns results from multiple HTTP requests, has a 1 second timeout length. Any arg names, must give in between 2 to 50 urls. (Ex: a=https://google.com)
